## 📚 Teoría Fundamental

### 4.1 Ecuaciones de Primer Grado (Lineales)

Forma general: **ax + b = 0**, donde a ≠ 0

**Solución:**
$$x = -\frac{b}{a}$$

#### Pasos para resolver:
1. Eliminar paréntesis y denominadores
2. Pasar términos con x a un lado, constantes al otro
3. Reducir términos semejantes
4. Despejar x
5. Verificar la solución

### 4.3 Ecuación Cuadrática

Forma general: **ax² + bx + c = 0**, donde a ≠ 0

**Fórmula general:**
$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

#### Discriminante (Δ = b² - 4ac)

| Δ | Tipo de raíces | Número de soluciones reales |
|---|---------------|----------------------------|
| Δ &gt; 0 | Reales y distintas | 2 |
| Δ = 0 | Reales e iguales (doble) | 1 |
| Δ &lt; 0 | Complejas conjugadas | 0 (reales) |

---

### 4.4 Propiedades de las Raíces

Si x₁ y x₂ son raíces de ax² + bx + c = 0:

**Suma de raíces:**
$$x_1 + x_2 = -\frac{b}{a}$$

**Producto de raíces:**
$$x_1 \cdot x_2 = \frac{c}{a}$$

**Ecuación a partir de raíces:**
$$x^2 - (x_1 + x_2)x + x_1 \cdot x_2 = 0$$

---

### 4.5 Interpretación Geométrica

La ecuación cuadrática **y = ax² + bx + c** representa una **parábola**:

| Elemento | Significado |
|----------|-------------|
| **Vértice** | Punto máximo (a&lt;0) o mínimo (a&gt;0) |
| **Eje de simetría** | x = -b/(2a) |
| **Intersección con y** | Punto (0, c) |
| **Intersecciones con x** | Raíces de ax² + bx + c = 0 |
| **Concavidad** | Arriba (a&gt;0), Abajo (a&lt;0) |

### 4.6 Ecuaciones Bicuadráticas

Forma: **ax⁴ + bx² + c = 0**

**Método de sustitución:**
- Sea y = x²
- La ecuación se convierte en: ay² + by + c = 0
- Resolver para y
- Luego x = ±√y (solo si y ≥ 0)

**Ejemplo:**
$$x^4 - 5x^2 + 4 = 0$$



---

### 4.7 Ecuaciones Recíprocas

**Tipo I** (coeficientes simétricos):
$$ax^4 + bx^3 + cx^2 + bx + a = 0$$

**Método:** Dividir entre x² y sustituir y = x + 1/x

**Tipo II** (coeficientes antisimétricos):
$$ax^4 + bx^3 - bx - a = 0$$

**Método:** Factorizar como (x² - 1)(algo) o dividir entre x²
