## 📚 Teoría Fundamental

### 1.1 Operaciones con Polinomios

Un **polinomio** es una expresión algebraica de la forma:
$$P(x) = a_nx^n + a_{n-1}x^{n-1} + ... + a_1x + a_0$$

#### Suma de Polinomios
Se suman los coeficientes de términos semejantes.

**Ejemplo:**
$$(3x^2 + 5x - 2) + (2x^2 - 3x + 7) = 5x^2 + 2x + 5$$

#### Producto de Polinomios
Se aplica la propiedad distributiva.

**Ejemplo:**
$$(x + 2)(x - 3) = x^2 - 3x + 2x - 6 = x^2 - x - 6$$

### 1.2 Ecuaciones Lineales

Una **ecuación lineal** tiene la forma:
$$ax + b = 0$$

**Solución:**
$$x = -\frac{b}{a}$$

#### Pasos para resolver:
1. Eliminar paréntesis (si los hay)
2. Pasar términos con x al lado izquierdo
3. Pasar números al lado derecho
4. Despejar x
5. Verificar la solución

---

### 1.3 Factorización

| Método | Forma | Ejemplo |
|--------|-------|---------|
| Factor común | $ab + ac = a(b+c)$ | $6x + 9 = 3(2x + 3)$ |
| Diferencia de cuadrados | $a^2 - b^2 = (a+b)(a-b)$ | $x^2 - 16 = (x+4)(x-4)$ |
| Trinomio cuadrado perfecto | $a^2 + 2ab + b^2 = (a+b)^2$ | $x^2 + 6x + 9 = (x+3)^2$ |
| Trinomio de forma $x^2 + bx + c$ | Buscar dos números que sumen b y multipliquen c | $x^2 + 5x + 6 = (x+2)(x+3)$ |
